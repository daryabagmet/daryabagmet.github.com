Graduation project

A route-building program system for low mobility group combines machine learning tools, such as object recognition using computer vision, and route search algorithms to create convenient navigation tools for users. The architecture of this system is represented by a service-oriented architecture. The route-building system consists of a navigation system, an image processing and analysis system, and an image acquisition system. The following tools were used for implementation: JavaScript, React.js, Roboflow platform (pre-trained model), Here Technologies mapping service.
